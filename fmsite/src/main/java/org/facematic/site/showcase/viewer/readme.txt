Здесь показана конструкция правой стороны страницы на которую вы смотрите 
(TabSheet с закладками readme.txt, ShowCaseViewer.xml, ShowCase.java, ShowFiles.java)

Разметка ShowCaseViewer.xml предписывает создание экземпляра TabSheet с именем "showCaseViewer"

Метод ShowCaseViewer.init добавляет в TabSheet "showCaseViewer" необходимые закладки.
ShowCase.java и ShowFiles.java - аннотации, значения полей которых используются 
при динамическом конструировании контента ShowCaseViewer и ShowCaseStudio.
